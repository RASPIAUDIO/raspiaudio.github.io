Features of Home Assistant

  • Open-source platform for controlling and automating devices via a single interface
  • Supports lights, thermostats, cameras, sensors, and much more
  • Powerful automation engine for scripts and event-based rules
  • Voice-assistant integration (Amazon Alexa, Google Assistant, etc.)
  • Customisable UI with community add-ons and integrations
  • Built-in rules engine and event bus

What ESP Muse Adds

  • Play, pause, and volume control for media
  • Multi-room, synchronised audio playback
  • Playlist and queue management
  • Library browsing and search
  • Voice control through Alexa/Google Assistant integrations
  • Full control from the Home Assistant web UI & mobile app
  • Automations that start/stop media on conditions or events
  • Seamless integration with other Home Assistant components
